_generated\FunctionalTesterActions::grabDataFromResponseByJsonPath

[!] Method is generated. Documentation taken from corresponding module.

Returns data from the current JSON response using [JSONPath](http://goessner.net/articles/JsonPath/) as selector. JsonPath is XPath equivalent for querying Json structures. Try your JsonPath expressions [online](http://jsonpath.curiousconcept.com/). Even for a single value an array is returned. This method **require [`flow/jsonpath` > 0.2](https://github.com/FlowCommunications/JSONPath/) library to be installed**. Example: ``` php <?php // match the first `user.id` in json $firstUserId = $I->grabDataFromResponseByJsonPath('$..users[0].id'); $I->sendPUT('/user', array('id' => $firstUserId[0], 'name' => 'davert')); ?> ```
