_generated\AcceptanceTesterActions::canSeeElement

[!] Method is generated. Documentation taken from corresponding module.

Checks that the given element exists on the page and is visible. You can also specify expected attributes of this element. ``` php <?php $I->seeElement('.error'); $I->seeElement('//form/input[1]'); $I->seeElement('input', ['name' => 'login']); $I->seeElement('input', ['value' => '123456']); // strict locator in first arg, attributes in second $I->seeElement(['css' => 'form input'], ['name' => 'login']); ?> ```
